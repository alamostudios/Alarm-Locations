# Alarm-Locations

This repository is a public collection of Alarm Locations created for Fire Alarm Reborn (FAR).  
FAR is a paid FiveM resource available for purchase [here](https://store.inferno-collection.com/package/fire-alarm-reborn).

An Alarm Location (also called an Alarm System) is an entry in a Lua file, created using the [FAR Tool](https://docs.inferno-collection.com/resources/fire-alarm-reborn/developers/tool), that enables an interior/MLO/map (base game or custom) to be used by FAR. A single Lua file can contain multiple Alarm Locations. These Lua files, called Draft Files as they can be loaded as Drafts by the [FAR Tool](https://docs.inferno-collection.com/resources/fire-alarm-reborn/developers/tool), can be read by FAR when the resource is started to dynamically load Alarm Systems.

***

If you run a server using FAR and want to add Alarm Locations from this resource, ensure you only use Alarm Locations for interiors/MLOS/maps that are in your server, or else you will see floating/incorrect FAR props.

If your interiors/MLOs/maps are not listed here, or you want to customize existing ones to your liking, use the easy-to-use [FAR Tool](https://docs.inferno-collection.com/resources/fire-alarm-reborn/developers/tool).

***

If you are contributing to this repository, thank you! Alarm Locations should be categorized by author, and then map name - for example: `"inferno_chris/inferno_headquarters/"`, then the Alarm Location file.
